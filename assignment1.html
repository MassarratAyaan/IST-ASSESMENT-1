Assignment 1: OPTION TOPIC 7: Internet and Website Development

1. Introduction to the World Wide Web (WWW):

The World Wide Web (WWW), also more commonly known as the web, is the place in the internet where people upload websites and webpages. These webpages/websites are linked with a hypertext link which allows anybody to find them.

What is a Web Server?

A web server is a computer that runs websites. They store and process the websites, then deliver them when somebody requests them.

What are Web Browsers?

Web browsers are software used to veiw the world wide web. When a user wants to see a certain site, the browser will get the necesarry content and show it on the users screen.

2. How is the world wide web different to the internet?

The internet is a network of networks, connecting millions of computers around the world. As long as these computers are conected to the internet, they can communicate with one another. The WWW on the other hand, is a part of the internet, which allows users to veiw billions of different websites and webpages.

3. What it an intranet?

An intranet is a local or restricted network, only accessible to people who have authorization. Users can share organize, and store information on it.

How does it relate to the internet?

An intranet is like a mini internet, but instead of to the world, it is accessible to only a specific organization.

4. What is an internet protocol?

An interent protocol (IP) is the set of rules that allow devices in the same network to connect with eachother. This allows them to communicate, regardles of internal process, stucture, or design.

TCP/IP, HTTP, and FTP

IP is what the internet uses to move traffic from one network to another, which works with another protocol called TCP (Transmission Control Protocol), which makes reliable end-to-end delivery of messages.

HTTP (Hypertext Transfer Protocol) is a protocol which requests and returns web recources. This is fast but unreliable, as it does not check if the resources have been successfully delivered.

(FTP) File Transfer Protocol is a protcol which transfers files from one platform to another. This is slower than HTTP, but is reliable, as it resends the files if they arent sent.
